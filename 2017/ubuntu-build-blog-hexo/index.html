最新ubuntu-16.04.3搭建hexo发布博客

发表于 2017-09-15 | 分类于 教程
字数统计: 762 | 阅读时长 ≈ 3

一、 准备的工作

  1. 安装ubuntu-16.04.3 下载。在虚拟机或者电脑上装都可以，一路下一步这个没啥难度。

  2. 装好后执行下面代码

  1. 安装Node.js

查看 node 版本看是否装好

  1. NPM更换淘宝源：

如果想把Nodejs升级到最新版：

更改NodeJs环境变量：
sudo gedit /etc/profile
在打开的文件最下面添加如下：
export NODE_HOME=/usr/local/n/versions/node/8.2.1
export PATH=$PATH:$NODE_HOME/bin
export NODE_PATH=$NODE_HOME/lib/node_modules

  1. 安装git
    查看 git 版本

生成SSH Key

看是否连接地上
ssh -T git@github.com

二、 安装hexo

初始化博客文件夹
git clone https://github.com/aooom/aooom.github.io.git
hexo init blog

切换到该路径

cd blog

安装hexo的扩展插件

用gedit打开package.json看看里面列出的插件就不用装

安装其它插件

换主题
git clone https://github.com/iissnan/hexo-theme-next themes/next

生成html

hexo g 或 hexo generate

启动本地服务器

hexo s 或 hexo server

这一步之后就可以通过http://localhost:4000 查看了

问题：为什么访问http://localhost:4000/，无反应？
解决方法：可能是由于端口问题引起的。使用Ctrl+C中断本地服务，使用命令hexo s -p 5000重新开启本地服务，访问http://localhost:5000/可以看到博客页面了

#常见问题

hexo deploy没有反应？

修改配置文件：_config.yml时，冒号后面没加空格。
hexo s 网站打不开？

端口占用，换个端口就好了。执行命令hexo s -p 5000，并在浏览器地址栏输入http://localhost:5000，回车访问。
如何换主题？

将主题下载后，放到themes文件夹中即可。例如，下面命令安装next主题：git clone https://github.com/iissnan/hexo-theme-next themes/next。
